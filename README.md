# 🤖 AI News Summarizer (n8n)

## 🚀 Overview
This project is an AI-powered workflow built using n8n.

It fetches news from RSS feeds, summarizes them using AI, and sends automated email updates.

## 🔧 Features
- RSS Feed Integration
- AI Summarization (Gemini)
- Email Automation
- Social Media Content Creation

## 🛠 Tech Used
- n8n
- Gemini AI
- APIs
- Email services

## 📂 How to Use
1. Download JSON file
2. Import into n8n
3. Add your API keys
4. Run workflow
